# Advanced Terminal Tracker

A command-line task and habit manager built in Python with SQLite.

## Commands
* `python main.py add "Task Name" --priority 1`
* `python main.py list`
* `python main.py done [ID]`
* `python main.py search "query"`
* `python main.py habit "Drink Water"`
* `python main.py log [HABIT_ID]`
* `python main.py habits`
* `python main.py export-json`
